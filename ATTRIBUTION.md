# Asset Attribution

Clipping Club uses public-domain/open-access image assets downloaded from:

- The Metropolitan Museum of Art Collection API — public domain objects  
  https://metmuseum.github.io/
- Art Institute of Chicago API — public domain artworks  
  https://api.artic.edu/docs/

Images were converted into local PNG clipping assets under `assets/real-clippings/` and listed in `assets/real-clippings/manifest.json` with source URLs and credits where available.

Additional original Pulp-made clipping packs can be added later: scanned packaging, signs, fake ads, receipts, local texture photos, product photography, and cleared archive fragments.
